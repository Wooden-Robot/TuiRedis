# 更新记录

本文件记录项目的主要变更。

## [0.7.0] - 2026-03-14

### 新增
- 新增 Redis Sentinel 主库发现支持，TUI 连接页和 CLI 都可直接配置。
- 新增多 Sentinel 节点支持，可通过 `sentinel_nodes` 配置或重复传入 `--sentinel-node`。
- 新增 Sentinel 故障切换感知重试，在连接中断、超时或出现 `READONLY` 响应后可自动重连并重试常见操作。
- 新增显式 Redis Cluster 连接模式，基于 `RedisCluster` 访问集群。
- 新增 cluster 感知的 Key 扫描，以及聚合后的服务器信息、keyspace 和 dbsize 摘要。
- 新增主界面异步加载提示与过期响应丢弃保护，覆盖 Key 加载、分页、服务器信息和详情请求。
- 新增更安全的连接配置保存机制，默认不持久化密码，并补齐落盘失败提示。
- 新增中英文 changelog 文件。

### 调整
- 调整单机模式下的数据库处理逻辑，不再假设只有 `0..15`，而是动态识别服务端配置。
- 调整主界面数据读取路径，将阻塞型 Redis 操作移出 Textual UI 线程。
- 调整 cluster 模式行为：禁用 DB 切换、拒绝原生命令中的 `SELECT`，并在 Server Info 中明确标注聚合视图。
- 调整 sentinel 模式提示，明确其为控制平面连接，并在 UI 中展示对应限制。
- 调整 `iredis` 启动流程，先检查可执行文件是否存在。
- 调整连接配置去重规则，将凭据和 Redis 模式相关字段纳入比较。

### 修复
- 修复 `SELECT` 失败时 UI 仍错误提示切库成功的问题。
- 修复 `iredis` 连接 URL 在密码包含保留字符时的解析错误。
- 修复连接配置保存/删除失败却在 UI 上表现为成功的问题。
- 修复多处异步请求竞争导致旧结果覆盖新界面的情况。
- 修复 `RedisClient` 在部分异常测试和清理路径中因未完全初始化而报错的问题。
